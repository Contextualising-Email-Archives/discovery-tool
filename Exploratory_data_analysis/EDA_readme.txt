This folder consists of a sample file to do data analysis on the Enron data


Instructions to run EDA.py
1. This file should run after the Data_pre_processing/Enron_data.py
2. Run it in an ipython environment such as spyder to see the visualisation
2. System will ask for the dataPath. Enter path where the Enron.pkl is stored. For example, Your/path/Enron.pkl
3. System will display names of the custodians and the number of email files in each of their mail boxes.
